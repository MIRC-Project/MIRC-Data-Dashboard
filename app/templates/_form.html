How can I view the data?

All server-side data we have recorded can be accessed by entering a year, month and day in the three text fields below:

{{ form.hidden_tag() }}
{{ form.year.label }} {{ form.year( size = 4, class = "form-control" + (" is-invalid" if form.year.errors else "") + " rounded-0 shadow-none", **{"placeholder": "eg. 2022", "autocomplete": "off"}) }} {{ form.month.label }} {{ form.month( size = 4, class = "form-control" + (" is-invalid" if form.month.errors else "") + " rounded-0 shadow-none", **{"placeholder": "eg. 08", "autocomplete": "off"}) }} {{ form.day.label }} {{ form.day( size = 4, class = "form-control" + (" is-invalid" if form.day.errors else "") + " rounded-0 shadow-none", **{"placeholder": "eg. 02", "autocomplete": "off"}) }} {{ form.submit(class = "form-control btn btn-dark") }}

Notes
  • On integers must be entered into these text fields
  • The acceptable format for Year is YYYY eg. 1990 - 2022
  • The acceptable format for Month is MM eg. 01 - 12
  • The acceptable format for Day is DD eg. 01 - 31
  • Some YYYY-MM-DD combinations may result in blank graphs which means that no clients performed an internet measurement to the server
